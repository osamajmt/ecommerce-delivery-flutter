# Ecommerce Delivery App (Flutter)

Delivery application for drivers to manage assigned orders.

## Features
- Driver authentication
- View assigned orders
- Update order status (picked up, delivered)
- API integration

## Tech Stack
- Flutter
- Dart
- REST API
- GetX
